Sobre

Neste site disponibilizo minhas notas de aula em formato HTML e PDF. As notas estão predominante escritas em linguagem de marcação LaTeX.

Buscando fomentar a educação, as notas estão disponíveis sob licença CC-BY-SA 4.0. Além disso, você pode acessar os códigos-fonte do material no seguinte repositório GitHub https://github.com/phkonzen/notas.

Aproveito para agradecer aos(às) estudantes e colegas que de modo assíduo ou esporádico contribuem com sugestões, críticas e correções! :)
