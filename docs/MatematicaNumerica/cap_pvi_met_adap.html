7.3 Método adaptativo com controle de erro

Consideremos um problema de valor inicia

y′⁢(t) =f⁢(t,y⁢(t)),t>t0, (7.72)
y⁢(t0) =y0. (7.73)

e um método de passo simples

y(1) =y0, (7.74)
y(i+1)⁢(h(i+1)) =y(i)+h(i+1)⁢Φ⁢(t(i),y(i);h(i+1)), (7.75)

com t(i)=t0+(i-1)⁢h(i). Nesta seção, discutiremos uma estimava para o maior valor de h(i+1) tal que o erro de discretização global e⁢(t(i+1);h(i+1)) seja controlado por uma dada tolerância T⁢O⁢L, i.e.

|e⁢(t(i+1);h(i+1))|:=|y(i+1)⁢(h(i+1))-y⁢(t(i+1))|≈T⁢O⁢L. (7.76)

Para um método de ordem hp, pode-se mostrar que (veja, [3, Cap. 7, Seç. 7.2])

y(i+1)⁢(h(i+1))=y⁢(t(i+1))+ep⁢(t(i+1))⁢(h(i+1))p, (7.77)

onde e⁢(t(i+1)) é uma função apropriada. Então, assumindo que e⁢(t(i);h(i))=0, temos

ep⁢(t(i+1))=h(i+1)⁢ep′⁢(t(i)) (7.78)

e, portanto, para termos (7.76) impomos que

|(h(i+1))p+1⁢ep′⁢(t(i))|=T⁢O⁢L. (7.79)

Daí, se obtermos uma aproximação para ep′⁢(t(i)) teremos uma aproximação para o passo h(i+1).

Para estimarmos ep⁢(t(i+1)), observamos que de (7.77) temos

y(i+1)⁢(h(i+1)2)=y⁢(t(i+1))+ep⁢(t(i+1))⁢(h(i+1))p2p (7.80)

e, então, subtraindo esta de (7.77) temos

y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)=ep⁢(t(i+1))⁢(h(i+1)2)p⁢(2p-1), (7.81)

donde

ep⁢(t(i+1))⁢(h(i+1)2)p=y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)2p-1. (7.82)

Daí, de (7.78), obtemos

ep′⁢(t(i))⁢h(i+1)⁢(h(i+1)2)p=y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)2p-1, (7.83)

o que nos fornece a seguinte aproximação de ep′⁢(t(i))

ep′⁢(t(i))=1(h(i+1))p+1⁢2p2p-1⁢[y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)]. (7.84)

Assim sendo, de (7.79) temos que o passo h(i+1) apropriado é tal que

2p2p-1⁢|y(i+1)⁢(h(i+1))-y(i+1)⁢(h(i+1)2)|≈T⁢O⁢L. (7.85)

Com base nesta estimativa podemos propor o seguinte método de passo adaptativo. Partindo de uma escolha arbitrária de h, computamos y(i+1)⁢(h) e y(i+1)⁢(h/2) de y(i). Então, enquanto

2p2p-1⁢|y(i+1)⁢(h)-y(i+1)⁢(h2)|>T⁢O⁢L, (7.86)

tomamos sucessivas divisões de h por 2, até satisfazermos (7.85). Obtido o h que satisfaz (7.85), temos computado y(i+1) com h(i+1)=h.

Exemplo 7.3.1.

Consideremos o seguinte problema de valor inicial

y′-y =sen⁡(t),t>0 (7.87)
y⁢(0) =12. (7.88)

A Figura 7.2 mostra a comparação entre y⁢(t) e a solução numérica obtida da aplicação do método de Euler com passo adaptativo. No método, utilizamos o passo inicial h(1)=0,1 e tolerância T⁢O⁢L=10-4. Ao compararmos esta figura com a Figura (7.1) fica evidente o controle do erro.

O algoritmo utilizado neste exemplo pode ser implementado no GNU Octave com o seguinte código:

f = @(t,y) y+sin(t);

TOL=1e-4;
h=1e-1;
tf=1;

t0=0;
y0=0.5;

t=[];
y=[];

c=1;
do

  h = min(h,tf-t0);

  do
    #passo h
    y1=y0+h*f(t0,y0);
    #passo h/2
    y2=y0+h/2*f(t0,y0);
    y2=y2+h/2*f(t0+h/2,y2);
    #verifica TOL
    est = 2*abs(y1-y2);
    if (est > TOL)
      h/=2;
      if (h<1e-8)
        error("h muito pequeno")
      endif
    else
      t0+=h;
      y0=y2;

      t(c)=t0;
      y(c)=y0;
      c+=1;
    endif
  until ((est <= TOL))

until (abs(t0-tf)<1e-14)

ya = @(t) exp(t)-sin(t)/2-cos(t)/2;
printf("%1.1E %1.5E %1.1E\n",t0,y0,abs(y0-ya(1)))

plot(t,ya(t),’b-’,t,y,’r-’);grid
