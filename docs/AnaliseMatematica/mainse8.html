4.3 Continuidade uniforme

Deﬁnição 4.3.1. (Continuidade uniforme) Uma função f : D → ℝ, y = f(x), é dita ser uniformemente contínua se, dado qualquer 𝜀 > 0 existe δ > 0 tal que

x,y ∈ D,|x − y| < δ ⇒|f(x) − f(y)| < 𝜀. (4.7)

Exemplo 4.3.1. Vejamos os seguintes casos:

a)
f(x) = x é uniformemente contínua. De fato, consideremos x,y > 0 e |x − y| < δ para δ > 0 arbitrário. Então, se y < δ temos x < y + δ < 2δ e
|x −y|≤x + y < 2δ + δ < 3δ. (4.8)

Agora, se y >= δ, então

|x −y| = |x − y| x + y < δ y < δ δ = δ. (4.9)

Logo, em qualquer um dos casos |x −y| < 3δ. Por tanto, dado 𝜀 > 0, podemos escolher δ = 𝜀2∕9 de forma que

x,y > 0,|x − y| < δ ⇒|x −y| < 𝜖, (4.10)

o que conclui o resultado.

b)
A função f(x) = 1∕x não é uniformemente contínua. De fato, basta observar que, para qualquer escolha de δ > 0, temos
1 x − 1 x + δ = δ x2 + δx → +∞comx → 0. (4.11)

Teorema 4.3.1. (de Heine) Se f : [a,b] → ℝ é contínua em [a,b] =: I, então f é uniformemente contínua.

Demonstração.

Suponhamos, por contradição, que f não é uniformemente contínua. Então, para algum 𝜖 > 0 existem xn,yn ∈ I tal que

|xn − yn| < 1 ne|f(xn) − f(yn)| > 𝜖, (4.12)

para todo n ∈ ℕ. Agora, como (xn)n é uma sequência limitada, pelo teorema de Bolzano-Weierstrass ela possui uma subsequência convergente. Seja, então, (xn′)n′ uma tal subsequência e c o seu limite. Como xn′ ∈ [a,b] para todo n′, temos c ∈ [a,b]. Além disso, como |xn′ − yn′|→ 0, temos yn′ → c. Também, pela continuidade de f, temos f(xn′) → f(c) e f(yn′) → f(c). Logo, |f(xn′) − f(yn′)|→ 0, o que é um absurdo.
