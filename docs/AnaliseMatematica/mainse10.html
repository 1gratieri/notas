5.2 Regras operacionais

Teorema 5.2.1. Regras operacionais Se f : D → ℝ e g : D → ℝ são funções diferenciáveis no ponto x ∈ D, então também são deriváveis no mesmo ponto as funções f + g, kf, fg e, no caso de g(x)≠0, f∕g. Aĺém disso, temos:

(f + g)′(x) = f′(x) + g′(x), (kf)′(x) = kf′(x), (fg)′(x) = f′(x)g(x) + f(x)g′(x), (f∕g)′(x) = f′(x)g(x) − f(x)g′(x) g2(x) ,se g(x)≠0. (5.9)

Demonstração.

Deixaremos como exercício a demonstração para as funções f + g e kf (veja, Exercício 5.2.1). Para o caso de fg, basta observar que:

lim h→0(fg)(x + h) − (fg)(x) h = lim h→0f(x + h)g(x + h) − f(x)g(x) h = lim h→0f(x + h)g(x + h) − f(x)g(x + h) + f(x)g(x + h) − f(x)g(x) h = lim h→0f(x + h) − f(x) h g(x + h) + lim h→0f(x)g(x + h) − g(x) h = f′(x)g(x) + f(x)g′(x). (5.10)

Agora, para mostrar que f∕g é diferenciável no ponto x ∈ D basta mostrar para o caso de f ≡ 1. De fato,

lim h→0 1 h 1 g(x + h) − 1 g(x) = lim h→0 − g(x + h) − g(x) h 1 g(x + h)g(x) = −g′(x) g2(x). (5.11)

Teorema 5.2.2. (Regra da cadeia) Sejam g : Dg → ℝ derivável no ponto x ∈ Dg, f : Df → ℝ com g(Dg) ⊂ Df e derivável no ponto y = g(x). Nestas condições, a função composta f ∘ g é diferenciável no ponto x e (f ∘ g)′(x) = f′(g(x))g′(x).

Demonstração.

Como f é derivável em y = g(x), temos

f(y + k) − f(y) k = f′(y) + η(k), (5.12)

com η(k) → 0 quando k → 0. Rearranjando temos

f(y + k) − f(y) = k[f′(y) + η(k)] (5.13)

inclusive para k = 0. Agora, para todo h suﬁcientemente pequeno, pomos k = g(x + h) − g(x) e, então

f(g(x + h)) − f(g(x)) h = f(y + k) − f(y) h = [f′(y) + η(k)]k h = [f′(y) + eta(k)]g(x + h) − g(x) h → f′(g(x))g′(x),com h → 0. (5.14)

Teorema 5.2.3. (Derivada da função inversa) Seja f : I = (a,b) → ℝ, y = f(x), uma função derivável em I com f′(x) sempre positiva ou sempre negativa. Então, sua inversa x = f−1(y) é derivável no intervalo J = f(I) e (f−1)′(y) = 1∕f′(g(x)).

Demonstração.

Sejam y,y0 ∈ J, x = f−1(y) e x0 = f−1(y 0). Notemos que

f−1(y) − f−1g(y 0) y − y0 = x − x0 f(x) − f(x0) = f(x) − f(x0) x − x0 −1. (5.15)

Agora, basta observar que quando y → y0, temos x → x0 pela continuidade da f−1. Logo, tomando o limite nas expressões acima, temos o resultado desejado.
