7.2 Algumas consequências da convergência uniforme

Teorema 7.2.1. Seja fn : D → ℝ uma sequência de funções contínuas. Se fn converge uniformemente para uma função f : D → ℝ, então f é contínua.

Demonstração.

Primeiramente, observemos que para quaisquer x,a ∈ D e n ∈ ℕ, temos

|f(x) − f(a)| = |f(x) − fn(x) + fn(x) − fn(a) + fn(a) − f(a)| ≤|f(x) − fn(x)| + |fn(x) − fn(a)| + |fn(a) − f(a)|. (7.14)

Seja, então, 𝜀 > 0. Como fn converge uniformemente para f, existe N ∈ ℕ tal que, para todo n > N temos

|f(x) − fn(x)| < 𝜀 3e|fn(a) − f(a)| < 𝜀 3. (7.15)

Fixemos um n > N. Como fn é contínua, para cada a ∈ D, existe δ > 0 tal que

x ∈ D,|x − a| < δ ⇒|fn(x) − fn(a)| < 𝜀 3. (7.16)

Portanto, usando a desigualdade em (7.14), vemos que para cada a ∈ D, existe δ > 0 tal que

x ∈ D,|x − a| < 𝜀 ⇒|f(x) − f(a)| < 𝜀. (7.17)

Isto mostra a continuidade de f.

Teorema 7.2.2. Seja fn : [a,b] → ℝ uma sequência de funções contínuas. Se fn converge uniformemente para uma função f : [a,b] → ℝ, então

lim ∫ abf n(x)dx = ∫ ab[lim f n(x)]dx = ∫ abf(x)dx. (7.18)

Demonstração.

Seja 𝜀 > 0. Como fn converge uniformemente para f, temos que existe N ∈ ℕ tal que

x ∈ [a,b],n > N ⇒|fn(x) − f(x)| < 𝜀 (b − a). (7.19)

Além disso, do teorema anterior (Teorema 7.2.1), temos que f é contínua em [a,b] e, portanto, assim como fn, é integrável neste intervalo (veja Teorema 6.2.1). Por tudo isso, temos

n > N ⇒ ∫ abf n(x)dx −∫ abf(x)dx = ∫ ab|f n(x) − f(x)|dx ≤ 𝜀 (b − a)(b − a) = 𝜀. (7.20)

O que concluí a demonstração.

Teorema 7.2.3. Seja fn : [a,b] → ℝ uma sequência de funções com derivadas contínuas em [a,b], tal que fn′ converge uniformemente para uma função g : [a,b] → ℝ. Suponhamos, ainda, para algum ponto c ∈ [a,b], fn(c) é uma sequência convergente. Então, fn converge uniformemente para uma função f para a qual f′ = g, i.e.

d dx lim fn(x) = lim d dxfn(x). (7.21)

Demonstração.

Do teorema fundamental do cálculo (Teorema 6.3.2), para todo x ∈ [a,b] temos

fn(x) = fn(c) + ∫ cxf n′(t)dt. (7.22)

Como, por hipótese, fn(c) é convergente e fn′ é uniformemente convergente para a função g, temos, do teorema anterior (Teorema 7.2.2), que tomando o limite de n →∞ nesta última equação, obtemos

f(x) = f(c) + ∫ cxg(t)dt. (7.23)

Observemos que f′ = g. Fica de exercício, mostrar que fn converge uniformemente para f, o que completa a demonstração.
