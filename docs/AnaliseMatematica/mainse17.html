7.3 Séries de funções

Deﬁnição 7.3.1. (Séries de funções) Sejam dadas funções fi : D → ℝ, i = 1, 2, 3,…. A série das funções fi é a sequência das somas parciais

∑ i=1nf i(x) := f1(x) + f2(x) + f3(x) + ⋯ + fn(x). (7.24)

Comumente, denotamos uma tal série por

∑ i=1∞f i(x). (7.25)

Tendo em vista de uma série de funções ∑ ifi(x) é uma sequência de funções, os conceitos de convergência de séries são aplicações diretas das deﬁnições de convergência de sequências de funções. Ou seja, dizemos que uma série de funções é pontualmente convergente para a função f : D → ℝ quando, para cada 𝜀 > 0 e para cada x ∈ D, existe N tal que

n > N ⇒ ∑ i=1nf i(x) − f(x) = ∑ i=n+1∞f i(x) < 𝜀. (7.26)

Ainda, dizemos uma série de funções é uniformemente convergente quando, para cada 𝜀 > 0, existe N tal que

x ∈ D,n > N ⇒ ∑ i=n+1∞f i(x) < 𝜀. (7.27)

Assim como as deﬁnições de séries de funções, os seguintes teoremas são consequências diretas dos relacionados a sequências de funções.

Teorema 7.3.1. (Critério de Cauchy) Uma série de funções ∑ fi(x), fi : D → ℝ, é uniformemente convergente se, e somente se, para qualquer 𝜀 > 0, existe N tal que

x ∈ D,p ∈ ℕ,n > N ⇒ ∑ i=n+1n+pf n+i(x) < 𝜀. (7.28)

Demonstração.

Do critério de convergência de Cauchy para sequências de funções, temos que ∑ fi(x) é uniformemente convergente se, e somente se, para qualquer 𝜀 > 0, existe N tal que

x ∈ D,n,m > N ⇒ ∑ i=1nf i(x) −∑ i=1mf i(x) < 𝜀. (7.29)

Agora, assumindo m > n s.p.g. e denotando p = m − n temos

∑ i=1nf i(x) −∑ i=1mf i(x) = ∑ i=1mf i(x) −∑ i=1nf i(x) = ∑ i=m+1nf i(x) = ∑ i=n+1mf i(x) = ∑ i=n+1n+pf i(x) . (7.30)

Teorema 7.3.2. Seja ∑ fi(x) uma série de funções fi : D → ℝ contínua em D. Se ∑ fi(x) converge uniformemente para f : D → ℝ, então f é contínua em D.

Demonstração.

Sejam

Sn(x) = ∑ i=1nf i(x) (7.31)

as somas parciais da série ∑ fi(x). Como a soma ﬁnita de funções contínuas em um conjunto D é uma função contínua no mesmo, temos que Sn é uma função contínua. Logo, como Sn → f uniformemente, temos do Teorema 7.2.1 que f é contínua em D.
