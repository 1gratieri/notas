6.2 Integrabilidade de funções contínuas

Teorema 6.2.1. Toda função f : [a,b] → ℝ contínua em [a,b] é integrável.

Demonstração.

Seja dado 𝜀 > 0. Pelo teorema de Heine, f é uniformemente contínua e, portanto, existe δ > 0 tal que

x,y ∈ I := [a,b],|x − y| < δ ⇒|f(x) − f(y)| < 𝜀. (6.4)

Seja, agora, Sn uma sequência arbitrária de somas de Riemann com norma tendo a zero quando n →∞, i.e.

Sn = ∑ i=1nf(ξ i)Δxi, (6.5)

com max 1≤i≤nΔxi → 0 quando n →∞. Queremos provar que existe I ∈ ℝ tal que

I = lim n→∞Sn (6.6)

independentemente da escolha dos pontos xi e ξi. Para tanto, iremos usar o critério de convergência de Cauchy. Para tanto, sejam

Sn := ∑ i=1nf(ξ i)Δxi (6.7)

a soma de Riemann para uma dada partição Pn := {a = x0 < x1 < x2 < ⋯ < xn = b} com |Pn| < δ e

SM := ∑ i=1Mf(η i)Δyi (6.8)

a soma de Riemann para um reﬁnamento PM := {a = y0 < y1 < y2 < ⋯ < yM = b} de Pn. Como PM é um reﬁnamento de Pn, cada subintervalo [xi−1,xi] é a união de certos subintervalos [yr−1,yr], …, [ys−1,ys] e, portanto Δxi = Δyr + ⋯ + Δys. Ainda, a diferença Sn − SM conterá, então, termos da forma

f(ξi) ∑ j=rsf(η i)Δyj = ∑ j=rs[f(ξ i) − f(ηj)]δyj. (6.9)

Agora, como |ξi − ηj| < δ, temos |f(ξi) − f(ηj)| < 𝜀 e, portanto

f(ξi) −∑ j=rsf(η j)Δyj < 𝜀∑ j=rsΔy j = 𝜀Δxi. (6.10)

Estendendo este resultado, temos

|Sn − SM|𝜀∑ i=1nΔx i = 𝜀(b − a). (6.11)

Por ﬁm, sejam Sn e Sm somas de Riemann correspondentes às partições Pn e Pm, respectivamente, com |Pn| < δ e |Pm| < δ. Ainda, seja PM um reﬁnamento de ambas partições. Então

|Sn − Sm|≤|Sn − SM| + |SM − Sm| < 2𝜀(b − a). (6.12)

Isto mostra que, dada uma sequência arbitrária de partições Pn com |Pn|→ 0 quando n →∞, então o limite das somas de Riemann Sn destas partições existe quando n →∞. Falta mostrar que este limite é único.

Sejam, agora, Sn e Tn diferentes sequências de somas de Riemann cujas partições têm norma tendendo a zero quando n →∞. Então, por exemplo, a sequência

S1,T1,S2,T2,S3,T3,…,Sn,Tn,… (6.13)

também é uma sequência de somas de Riemann cujas partições têm norma tendo a zero quando n →∞. Logo, pelo que mostramos acima, o limite desta sequência existe. Agora, como (Sn)n e (Tn)n são subsequências destas, elas convergem para o mesmo limite.

Exercícios

E 6.2.1. Mostre que se f : [a,b] → ℝ é integrável em [a,b], então

∫ abf(x)dx = ∫ acf(x)dx + ∫ cbf(x)dx, (6.14)

para qualquer c ∈ [a,b].
