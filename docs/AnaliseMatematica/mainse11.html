5.3 Extremos e o teorema do valor médio

Teorema 5.3.1. Se f : D → ℝ é derivável num ponto c ∈ D no qual ela assume valor máximo ou mínimo local, então f′(c) = 0.

Demonstração.

No caso de c ser ponto de mínimo local de f, então existe δ > 0 tal que f(c + h) − f(c) ≥ 0 para todo |h| < δ. Logo, temos

lim h→0+f(c + h) − f(c) h ≥ 0e lim h→0−f(c + h) − f(c) h ≤ 0. (5.17)

Mas, então, como f é diferenciável no ponto c, necessariamente f′(c) = 0. Um raciocínio análogo mostra o resultado para o caso de c ser ponto de máximo local (veja o Exercício 5.3.1.

■

Teorema 5.3.2. (de Rolle) Se f : [a,b] → ℝ é uma função contínua em todo o seu domínio, derivável no intervalo aberto (a,b) e f(a) = f(b), então existe c ∈ (a,b) tal que f′(c) = 0.

Demonstração.

Se f é constante, então f′(c) = 0 para todo c ∈ (a,b). Caso contrário, f terá que assumir valores maiores ou menores que f(a) = f(b). Como f é contínua no intervalo fechado [a,b] ela assumirá valor máximo ou mínimo em algum ponto c ∈ (a,b) (veja Teorema 4.2.2). Então, pelo Teorema 5.3.1, temos f′(c) = 0.

■

Teorema 5.3.3. (do valor médio) Se f : [a,b] → ℝ é uma função contínua em todo o seu domínio e derivável no intervalo (a,b), então existe c ∈ (a,b) tal que

f(b) − f(a) b − a = f′(c). (5.18)

Demonstração.

Seja

g(x) = f(x) − f(a) − f(b) − f(a) b − a (x − a). (5.19)

Observamos que g é contínua em [a,b], derivável em (a,b) e g(a) = g(b) = 0. Logo, pelo Teorema de Rolle, temos que existe c ∈ (a,b) tal que g′(c) = 0, mas daí f′(c) satisfaz o resultado desejado.

■

Exercícios

E 5.3.1. Mostre que se f : D → ℝ é derivável num ponto c ∈ D no qual ela assume valor máximo local, então f′(c) = 0.

E 5.3.2. Use o teorema do valor médio para mostrar que se f : [a,b] → ℝ é contínua em todo o seu domínio e f′(x) > 0 para todo x ∈ (a,b), então f é uma função crescente.

E 5.3.3. Mostre que se f : [a,b] → ℝ é diferenciável em (a,b) e f′ é limitada em (a,b), então f é uniformemente contínua.
