3.4 Limites no inﬁnito e limites inﬁnitos

Deﬁnição 3.4.1. (Limites inﬁnitos) Sejam f : D → ℝ, y = f(x), e a um ponto de acumulação de D. Dizemos que o limite de f(x) é + ∞ quando x → a se, para todo k > 0 existe δ > 0 tal que para todo x ∈ D, 0 < |x − a| < δ temos f(x) > k. Analogamente, dizemos que o limite de f(x) é −∞ quando x → a se, para todo k > 0 existe δ > 0 tal que para todo x ∈ D, 0 < |x − a| < δ temos f(x) < −k. Nestes casos escrevemos

lim x→af(x) = +∞ou lim x→af(x) = −∞, (3.12)

respectivamente.

Exemplo 3.4.1. Vejamos os seguintes casos:

a)
lim x→01∕|x| = +∞. De fato, dado k > 0 basta tomarmos δ = 1∕k. Com isso, 0 < |x − 0| < δ implica |x| < 1∕k e, portanto, 1∕|x| > k, i.e. |1∕|x|− 0| > k.
b)
Seja f : (−∞, 0) → ℝ, y = f(x) := 1∕x. Neste caso, lim x→0f(x) = −∞. Deixamos a veriﬁcação para o leitor.

Deﬁnição 3.4.2. (Limite no inﬁnito) Seja f : D → ℝ, y = f(x). Quando D é ilimitado superiormente dizemos que f(x) tende a L quando x → +∞ se, para todo 𝜀 > 0 existe k > 0 tal que x > k implica |f(x) − L| < 𝜀. Analogamente, quando D é ilimitado inferiormente dizemos que f(x) tende a L quando x →−∞ se, para todo 𝜀 > 0 existe k > 0 tal que x < −k implica |f(x) − L| < 𝜀. Nestes casos escrevemos

lim x→+∞f(x) = Lou lim x→−∞f(x) = L, (3.13)

respectivamente.

Exemplo 3.4.2. Vejamos os seguintes casos:

a)
lim x→∞1∕x = 0. De fato, dado 𝜀 > 0 escolhemos δ = 1∕𝜀. Com isso, x > δ implica 0 < 1∕x < 1∕δ = 𝜀 e, portanto, |1∕x − 0| < 𝜀.
b)
lim x→−∞1∕x = 0. Caso análogo ao anterior, veriﬁque!

Observação 3.4.1. Observe que deﬁnições análogas às 3.3.1, 3.4.1 e 3.4.2 se aplicam para os casos:

lim x→a+∕−f(x) = ±∕ ∓∞ou lim x→±∞f(x) = L. (3.14)

Também, consideramos deﬁnições análogas para os casos:

lim x→a+∕−f(x) = L±∕∓ou lim x→+∕−∞f(x) = L±∕∓. (3.15)

Teorema 3.4.1. Toda função monótona e limitada superiormente (inferiormente), cujo domínio contenha [c, +∞) ((∞,c]), possui limite quando x → +∞ (x →−∞).

Demonstração.

Consideremos o caso de f : [c, +∞) → ℝ, y = f(x), função não decrescente e limitada superiormente. Seja, então L o supremo do conjunto imagem de f. Mostraremos que f(x) → L quando x → +∞. De fato, dado 𝜀 > 0 existe k > 0 tal que L − 𝜀 < f(k) ≤ L. Agora, como f é não decrescente, para todo x > k temos L − 𝜀 < f(k) < f(x) ≤ k e, portanto, |f(x) − L| < 𝜀. Isto demostra o caso considerado e deixamos para o leitor a veriﬁcação dos demais (veja, também, Exercício 3.4.1).
