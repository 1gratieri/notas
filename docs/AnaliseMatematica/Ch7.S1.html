7.1 Sequência de funções

Definição 7.1.1.

Uma sequência de funções (fn)n∈ℕ é um conjunto de funções fn:D→ℝ, y=fn⁢(x), indexadas por n∈ℝ. Comumente, utiliza-se a notação (fn⁢(x))n∈ℕ (ou, simplesmente, fn⁢(x)) para explicitar que trata-se de uma sequência de funções.

Observação 7.1.1.

Salvo explicitado ao contrário, ao longo deste capítulo assumiremos que as funções que compõe uma dada sequência têm todas o mesmo domínio.

Exemplo 7.1.1.

Vejamos os seguintes exemplos:

  1. a)

    fn⁢(x)=x+1/n, n∈ℕ, é uma sequência de funções afins.

  2. b)

    gn⁢(x)=xn é uma sequência de polinômios.

  3. c)

    hn⁢(x)=1+x+x2+⋯+xn é, também, uma sequência11Um sequência deste tipo também é chamada de série de funções, como definiremos logo adiante no texto. de polinômios.

7.1.1 Limite pontual

Definição 7.1.2.

Limite pontual Diz-se que uma sequência de funções (fn⁢(x))n∈ℝ, fn:D→ℝ, converge pontualmente (ou simplesmente) para uma função f⁢(x), f:D→ℝ, se, dado qualquer ε>0, para cada x∈D, existe N tal que

n>N⇒|fn⁢(x)-f⁢(x)|<ε. (7.1)
Exemplo 7.1.2.

Vejamos os seguintes casos.

  1. a)

    A sequência de funções fn⁢(x)=x+1/n converge pontualmente para a função identidade f⁢(x)=x. De fato, sejam ε>0 e x no domínio da f. Escolhendo N>1/ε, temos

    n>N⇒|fn⁢(x)-f⁢(x)|=|x+1n-x|=|1n|<1N<ε. (7.2)
  2. b)

    A sequência de funções gn⁢(x)=x/n converge pontualmente para a função nula f⁢(x)≡0. De fato, sejam ε>0 e x no domínio da f. Escolhendo N>|x|/ε, temos

    n>N⇒|xn-0|<|x|n<|x|N<ε. (7.3)

7.1.2 Convergência uniforme

Definição 7.1.3.

Convergência uniforme Diz-se que uma sequência de funções (fn⁢(x))n∈ℝ, fn:D→ℝ, converge uniformemente para uma função f⁢(x), f:D→ℝ, se, dado qualquer ε>0, existe N tal que

x∈D,n>N⇒|fn⁢(x)-f⁢(x)|<ε. (7.4)
Exemplo 7.1.3.

Vejamos os seguintes casos:

  1. a)

    No Exemplo 7.1.2 a), vimos que a sequência de funções fn⁢(x)=x+1/n é pontualmente convergente para a função f⁢(x). Agora, observando a demonstração vemos que a convergência é também uniforme. Verifique!

  2. b)

    A sequência de funções fn⁢(x)=x/n é pontualmente convergente para f⁢(x)≡0, mas não é uniformemente convergente. De fato, dado ε>0 e x no domínio, escolhemos N>|x|/ε e, então, temos

    n>N⇒|fn⁢(x)-f⁢(x)|=|xn-0|≤|x|N<ε. (7.5)

    Isto mostra a convergência pontual. Entretanto, por exemplo, tomemos ε=1. Então, dado qualquer n∈ℕ escolhemos x=2/n. Logo, temos

    |fn⁢(x)-f⁢(x)|=|xn|=xn=2>ε. (7.6)

    Isto mostra que a convergência de fn→0 não é uniforme.

Teorema 7.1.1.

(Critério de convergência de Cauchy) Uma sequência de funções fn:D→ℝ é uniformemente convergente para uma função f:D→ℝ se, e somente se, dado qualquer ε>0, exista N∈ℕ tal que

x∈D,n,m>N⇒|fn⁢(x)-fm⁢(x)|<ε. (7.7)
Demonstração.

Mostraremos, separadamente, que a condição é necessária e suficiente.

  1. a)

    Necessidade. Seja ε>0. Por hipótese, fn⁢(x)→f⁢(x) uniformemente, ou seja, existe N tal que

    x∈D,n>N⇒|fn⁢(x)-f⁢(x)|<ε2. (7.8)

    Logo, para este mesmo N, temos

    x∈D,n,m>N⇒|fn⁢(x)-fm⁢(x)|≤|fn⁢(x)-f⁢(x)|+|fm⁢(x)-f⁢(x)|<ε. (7.9)

    Isto mostra que (fn) satisfaz o critério de Cauchy.

  2. b)

    Suficiência. Comecemos construindo nosso candidato a limite. Para cada x∈D, (fn⁢(x)) é uma sequência de números reais que, pela hipótese, satisfaz o critério de Cauchy e, portanto, fn⁢(x) converge quando n→∞. Seja, então, f:D→ℝ a função tal que f⁢(x) é o limite de fn⁢(x) para cada x∈D. Mostraremos, agora, que fn converge uniformemente para f. Seja dado ε>0. Por hipótese, existe N tal que

    x∈D,n,m>N⇒|fn⁢(x)-fm⁢(x)|<ε2. (7.10)

    Agora, observemos que fn⁢(x)-fm⁢(x)→fn⁢(x)-f⁢(x) pontualmente quando m→∞. Logo, passando ao limite na afirmação (7.10), temos

    x∈D,n>N⇒|fn⁢(x)-f⁢(x)|≤ε2<ε, (7.11)

    o que mostra a convergência uniforme.

∎

Exemplo 7.1.4.

No exemplo anterior (Exemplo 7.1.3 a)) vimos que fn⁢(x)=x+1/n converge uniformemente para f⁢(x)=x. Aqui, mostraremos que fn satisfaz o critério de Cauchy para sequência de funções. Seja ε>0. Observemos que 1/n→0 quando n→∞ e, portanto, satisfaz o critério de Cauchy para sequências de números. A saber, existe N tal que

n,m>N⇒|1n-1m|<ε. (7.12)

Daí, temos também que

x∈ℝ,n,m>n⇒|fn⁢(x)-fm⁢(x)|=|x+1n-x-1m|=|1n-1m|<ε. (7.13)

O que concluí que fn satisfaz o critério de Cauchy.

Exercícios

E 7.1.1.

Mostre que a sequência de funções fn:ℝ∖{0}, fn⁢(x)=1/(n⁢x), converge pontualmente para a função nula f⁢(x)≡0.

E 7.1.2.

Mostre que a sequência de funções fn⁢(x)=cos⁡(x/n) converge pontualmente para função constante f⁢(x)≡1.

E 7.1.3.

Mostre que a sequência de funções fn⁢(x)=e-(x-n)2 é pontualmente convergente para f⁢(x)≡0.

E 7.1.4.

Mostre que a sequência de funções fn⁢(x)=e-(x-n)2 não é uniformemente convergente para f⁢(x)≡0.

E 7.1.5.

Mostre que a sequência de funções fn:[-1,1]→ℝ, fn⁢(x)=ex/n, é uniformemente convergente para a função f⁢(x)≡1.

E 7.1.6.

Mostre que a sequência de funções fn⁢(x)=x2/(1+n⁢x2) satisfaz o critério de convergência de Cauchy para sequências de funções. Dica: observe que 0≤fn⁢(x)<1/n.
