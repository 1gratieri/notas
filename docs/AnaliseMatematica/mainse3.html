3.2 Limites

Deﬁnição 3.2.1. (Limite) Sejam uma função f : D → ℝ, y = f(x), e a um ponto de acumulação de D. Diz-se que L ∈ ℝ é o limite de f(x) com x tendendo a a se, para todo 𝜀 > 0, existe δ > 0 tal que

x ∈ D, 0 < |x − a| < δ ⇒|f(x) − L| < 𝜀. (3.1)

Quando isso ocorre, escrevemos

lim x→af(x) = L, (3.2)

ou ainda, simplesmente, f(x) → L quando x → a.

Exemplo 3.2.1. Vejamos os seguintes casos:

a)
Temos lim x→1x − 1 = 0. Isto segue imediatamente, pois, neste caso, f(x) = x − 1, a = 1, L = 0 e, então, dado 𝜀 > 0, tomamos δ = 𝜀 de forma que
x ∈ ℝ, 0 < |x − 1| < δ ⇒|x − 1 − 0| < 𝜀. (3.3)
b)
A função não precisa estar deﬁnida no ponto em o limite é tomado. Por exemplo, lim x→1x2−1 x+1 = 0. Veriﬁque!

Observação 3.2.1. Quando nos referirmos a expressão “x tende a a” (ou similares), estaremos sempre assumindo que a é um ponto de acumulação do domínio da função de interesse.

3.2.1 Propriedades do limite

Teorema 3.2.1. Se f : 𝔻 → ℝ, y = f(x), com lim x→af(x) = L, então lim x→a|f(x)| = |L|.

Demonstração.

Seja 𝜀 > 0. Por hipótese, existe δ > 0 tal que x ∈ D, 0 < |x − a| < δ implica |f(x) − L| < 𝜀. Tomando, então, um tal δ e observando que |f(x)|−|L| < |f(x) − L|, temos que para todo x ∈ D, 0 < |x − a| < δ, ocorre |f(x)|−|L| < 𝜀.

■

Teorema 3.2.2. Se f : 𝔻 → ℝ, y = f(x), com lim x→af(x) = L e A < L < B, então existe δ > 0 tal que x ∈ D, 0 < |x − a| < δ implica A < f(x) < B.

Demonstração.

De fato, por hipótese, para cada 𝜀 > 0, existe δ > 0 tal que x ∈ D, 0 < |x − a| < δ implica |f(x) − L| < 𝜀. Então, o resultado segue escolhendo um tal δ quando 𝜀 = min{L − A,B − L}.

■

Corolário 3.2.1. (Permanência do sinal) Se f : 𝔻 → ℝ, y = f(x), com lim x→af(x) = L > 0 (L < 0), então existe δ > 0 tal que x ∈ D, 0 < |x − a| < δ, implica f(x) > 0 (f(x) < 0).

Demonstração.

Quando L > 0 (L < 0) basta escolher A = 0 (B = 0) no teorema anterior.

■

Teorema 3.2.3. (Operações com limites) Sejam f1,f2 : D → ℝ, y = f1(x), y = f2(x), com lim x→af1(x) = L1 e lim x→af2(x) = L2, então (omitindo que x → a)

a)
lim[f1(x) + f2(x)] = lim f1(x) + lim f2(x).
b)
para todo k ∈ ℝ, temos lim kf1(x) = k lim f1(x).
c)
lim f1(x)f2(x) = lim f1(x) ⋅ lim f2(x).
d)
lim f1(x) f2(x) = lim f1(x) lim f2(x), quando L2≠0.

Demonstração.

Seja dado 𝜀 > 0.

a)
Seja δ > 0 tal que x ∈ D, 0 < |x − a| < δ implica |f1(x) − L1| < 𝜀∕2 e |f2(x) − L2| < 𝜀∕2. Logo, para tais δ e x temos
|(f1(x) + f2(x)) − (L1 + L2)|≤|f1(x) − L1| + |f2(x) − L2| < 𝜀 2 + 𝜀 2 = 𝜀. (3.4)
b)
O resultado é imediato para k = 0. Sejam k≠0 e δ > 0 tal que x ∈ D, 0 < |x − a| < δ implica |f1(x) − L1| < 𝜀∕|k|. Então, para tais δ e x temos |kf1(x) − kL1| = |k||f1(x) − L1| < |k|𝜀∕|k| = 𝜀.
c)
Sejam M > 0 e δ > 0 tal que x ∈ D, 0 < |x − a| < δ implica |f1(x) − L1| < 𝜀∕(2|L2|), |f1(x)| < M (veja Teorema 3.2.2) e |f2(x) − L2| < 𝜀∕(2M). Então
|f1(x)f2(x) − L1L2| = |f1(x)f2(x) − f1(x)L2 + f1(x)L2 − L1L2| = |f1(x)(f2(x) − L2) + (f1(x) − L1)L2| ≤|f1(x)||f2(x) − L2| + |f1(x) − L1||L2| < M 𝜀 2M + 𝜀 2|L2||L2| = 𝜀. (3.5)
d)
De c), basta mostrar que 1∕f2(x) → 1∕L2 quando x → a. Para tando, seja δ > 0 tal que x ∈ D, 0 < |x − a| < δ implica |f2(x) − L2| < 𝜀L22 2 e |f2(x)| > |L2|∕2 (veja Teorema 3.2.2). Então, para tais δ e x temos
1 f2(x) − 1 L2 = |f2(x) − L2| |f2(x)L2| < 𝜀L22 2 |L2||L2| 2 = 𝜀. (3.6)

Teorema 3.2.4. O limite de uma função f : D → ℝ é L quando x → a se, e somente se, para toda sequência (xn)n∈ℝ ⊂ D ∖{a} com xn → a, temos f(xn) → L.

Demonstração.

a)
Primeiramente, mostraremos que se lim x→af(x) = L, então dada qualquer sequência (xn)n∈ℝ ⊂ D ∖{a} com xn → a, temos f(xn) → L. De fato, sejam 𝜀 > 0 e (xn)n∈ℝ ⊂ D ∖{a} com xn → a. Então, por hipótese, existe δ > 0 tal que x ∈ D, 0 < |x − a| < δ implica |f(x) − L| < 𝜀. Agora, como xn → a, existe N suﬁcientemente grande tal que n > N implica |xn − a| < δ e, portanto, |f(xn) − L| < 𝜀. Ou seja, f(xn) → L.
b)
Aqui, provaremos por absurdo que se para toda sequência (xn)n∈ℝ ⊂ D ∖{a} com xn → a temos f(xn) → L, então lim x→af(x) = L. Ou seja, vamos assumir que existe um 𝜀 > 0 tal que para todo δ > 0 existe algum x ∈ D, 0 < |xn − a| < δ com |f(x) − L| > 𝜀. Sejam um tal 𝜀 e para cada n ∈ ℕ um xn′ ∈ D com 0 < |xn′ − a| < 1∕n e |f(xn′) − L| > 𝜀. Com isso, temos formado uma sequência (xn′) ⊂ D ∖{a}, xn′ → a, mas f(xn′)↛L.

■

Corolário 3.2.2. Um função f : D → ℝ, y = f(x), tem limite L quando x → a se, e somente se, para toda sequência (xn)n∈ℕ ⊂ D ∖{a} com xn → a temos que f(xn) n∈ℕ é convergente.

Demonstração.

Segue, imediatamente, do fato de que se (yn) é uma sequência com yn → L, então toda subsequência de (yn) é convergente e converge para L.

■

Teorema 3.2.5. (Critério de convergência de Cauchy) Uma condição necessária e suﬁciente para que uma função f : D → ℝ, y = f(x), tenha limite L quando x → a é que, para todo 𝜀 > 0, exista δ > 0 tal que

x,y ∈ V δ′(a) ∩ D ⇒|f(x) − f(y)| < 𝜀. (3.7)

Demonstração.

a)
A suﬁciência segue do critério de convergência de Cauchy para sequências e do Corolário 3.2.2.
b)
Exercício 3.2.4.

Exercícios

E 3.2.1. Diga se é verdadeira ou falsa a seguinte aﬁrmação: se f : 𝔻 → ℝ, y = f(x), com lim x→af(x) = 0, então f(a) = 0. Justiﬁque sua resposta.

E 3.2.2. Mostre que se f : 𝔻 → ℝ, y = f(x), com lim x→af(x) = L > 0, então lim x→a f(x) = L.

E 3.2.3. Demonstre o Teorema 3.2.3 como um corolário do Teorema 3.2.4.

E 3.2.4. Demonstre que se f : D → ℝ, y = f(x), tem limite L quando x → a, então para qualquer 𝜀 > 0, existe δ > 0 tal que x,y ∈ V δ′(a) ∩ D implica |f(x) − f(y)| < 𝜀.
