3.1 Noções de topologia

Deﬁnição 3.1.1. (Ponto interior) Diz-se que x é um ponto interior de um dado conjunto C quando existe um intervalo (a,b) que contém x e está contido em C, i.e. x ∈ (a,b) ⊂ C. O conjunto de todos os pontos interiores de C é chamado de seu interior.

Exemplo 3.1.1.

a)
Todo elemento de um intervalo aberto (a,b) é ponto interior deste.
b)
O interior de um dado intervalo fechado [a,b] é o intervalo aberto (a,b).

Deﬁnição 3.1.2. (Conjunto aberto) Diz se que C é conjunto aberto quando todos seus elementos são pontos interiores.

Exemplo 3.1.2. Vejamos os seguintes casos:

a)
O intervalo (a,b) := {x ∈ ℝ; a < x < b} é um conjunto aberto. De fato, dado x ∈ (a,b) podemos tomar 0 < 𝜀 < min{x − a,b − x} de forma que x ∈ (x − 𝜀,x + 𝜀) ⊂ (a,b).
b)
O intervalo (a,b] não é aberto, pois b ∈ (a,b] não é ponto interior.
c)
O conjunto vazio ∅ é um conjunto aberto. Com efeito, se o conjunto ∅ não é aberto, então existe um elemento x ∈∅ que não é ponto interior de ∅, o que é um absurdo pois ∅ não contém elementos por deﬁnição.
d)
O conjunto dos números racionais ℚ não é aberto.

Deﬁnição 3.1.3. (Vizinhança) Uma vizinhança de um dado ponto x é qualquer conjunto V que contenha x como ponto interior. Também, a vizinhança simétrica de um ponto x ∈ ℝ é todo intervalo V 𝜀(x) := (x − 𝜀,x + 𝜀) com 𝜀 > 0. Mais estrito, a vizinhança perfurada de x ∈ ℝ é uma vizinhança de x que não contém x. Aproveitamos para ﬁxar a notação:

V 𝜀′(x) := V 𝜀(x) ∖{x} = {y ∈ ℝ; 0 < |x − y| < 𝜀}.

Exemplo 3.1.3. Podemos reescrever o Exemplo 3.1.2 da seguinte forma. Um intervalo (a,b) é um conjunto aberto, pois para cada x ∈ (a,b) podemos escolher 0 < 𝜀 < min{x − a,b − x} tal que V 𝜀(x) ⊂ (a,b).

Deﬁnição 3.1.4. (Ponto de acumulação) Um ponto x é chamado de ponto de acumulação de um dado conjunto C quando toda vizinhança de x contém inﬁnitos pontos de C.

Exemplo 3.1.4. Vejamos os seguintes casos:

a)
O número a é ponto de acumulação do intervalo (a,b] não degenerado. De fato, dado 𝜀 > 0, temos (a,a + 𝜀) ⊂ V 𝜀(a) e (a,a + 𝜀) ∩ (a,b] é um conjunto inﬁnito.
b)
Zero é o único ponto de acumulação do conjunto {1, 1∕2, 1∕3,…, 1∕n,…}.

Deﬁnição 3.1.5. (Ponto isolado) Diz que x é ponto isolado de um dado conjunto C quando x ∈ C não é ponto de acumulação de C. Diz-se que um conjunto é discreto quando todos seus elementos são pontos discretos.

Exemplo 3.1.5. Vejamos os seguintes casos:

a)
O conjunto dos números naturais ℕ é discreto.
b)
O conjunto dos números racionais ℚ não é discreto.
c)
O conjunto {1, 1∕2, 1∕3,…, 1∕n,…} é discreto.

Deﬁnição 3.1.6. (Ponto aderente) Dizemos que x é ponto aderente de um dado conjunto C quando toda vizinhança de x contém algum ponto de C. O conjunto de todos os pontos aderentes de C é chamado de fecho (ou, conjunto de aderência) de C, o qual denotamos por C¯.

Observação 3.1.1. Observe que todo ponto de um conjunto é aderente ao mesmo, bem como, todos os seus pontos de acumulação.

Exemplo 3.1.6. Vejamos os seguintes casos:

a)
O fecho de (a,b] é o intervalo fechado [a,b].
b)
O conjunto dos números reais ℝ é o fecho do conjunto dos números racionais ℚ, i.e. Q¯ = ℝ.

Deﬁnição 3.1.7. Conjunto fechado Dizemos que um conjunto C é fechado quando é igual ao seu fecho, i.e. C = C¯.

Exemplo 3.1.7. Vejamos os seguintes casos:

a)
O intervalo [a,b] é um conjunto fechado.
b)
O conjunto vazio ∅ é fechado. Por quê?
c)
O conjunto dos números reais ℝ é fechado.
d)
O conjunto dos números racionais ℚ não é fechado.

Deﬁnição 3.1.8. (Conjunto denso) Dizemos que um conjunto A é denso no conjunto B, quando todo ponto aderente de A¯ ⊂ B.

Exemplo 3.1.8. O conjunto dos números racionais ℚ é denso no conjunto dos números reais ℝ.

3.1.1 Exercícios

E 3.1.1. Seja dado um conjunto C. Mostre que x é ponto de acumulação de C se, e somente se, toda vizinhança de x contém pelo menos um elemento de C diferente de x.

E 3.1.2. Seja dado um conjunto C. Mostre que x é ponto isolado de C se, e somente se, existe uma vizinhança de x tal que (V (x) ∖{x}) ∩ C = ∅.
