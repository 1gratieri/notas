3.3 Limites laterais

Deﬁnição 3.3.1. (Limite lateral) Sejam uma função f : D → ℝ, y = f(x), e a um ponto de acumulação de D. Dizemos que L ∈ ℝ é o limite de f(x) com x tendendo a a pela direita se, para todo 𝜀 > 0, existe δ > 0 tal que

x ∈ D, 0 < x − a < δ ⇒|f(x) − L| < 𝜀. (3.9)

Quando isso ocorre, escrevemos

lim x→a+f(x) = L, (3.10)

ou ainda, simplesmente, f(x) → L quando x → a+. Analogamente, escreve-se f(x) → L quando x → a−, ou lim x→a−f(x) = L se, para todo 𝜀 > 0, existe δ > 0 tal que

x ∈ D, 0 < a − x < δ ⇒|f(x) − L| < 𝜀. (3.11)

Exemplo 3.3.1. Vejamos os seguintes casos:

a)
lim x→0+ x |x| = 1. De fato, dado qualquer 𝜀 > 0 podemos escolher, por exemplo, δ = 1 e, com isso, para todo x ∈ ℝ, 0 < x − 0 < 1 implica |x∕|x|− 1| = 0 < 𝜀.
b)
lim x→0− x |x| = −1. De fato, dado qualquer 𝜀 > 0 podemos escolher, por exemplo, δ = 1 e, com isso, para todo x ∈ ℝ, 0 < 0 − x < 1 implica |x∕|x|− (−1)| = |− 1 + 1| = 0 < 𝜀.

Deﬁnição 3.3.2. Ponto de acumulação lateral Seja C um conjunto. Dizemos que a ∈ C é ponto de acumulação à esquerda de C se, para todo 𝜀 > 0 o conjunto (a − 𝜀,a) ∩ C contém inﬁnitos pontos de C. Analogamente, dizemos que a ∈ C é ponto de acumulação à direita de C se, para todo 𝜀 > 0 o conjunto (a,a + 𝜀) ∩ C contém inﬁnitos pontos de C.

Teorema 3.3.1. Se f : I → ℝ, y = f(x), é uma função monótona e limitada, deﬁnida em um intervalo I no qual a é ponto de acumulação à esquerda (ponto de acumulação à direita), então f tem limite com x → a− (x → a+).

Demonstração.

Consideremos o caso em que f é uma função não crescente e a seja ponto de acumulação à direita. Seja, então L o supremo do conjunto formado por f(x) com x ∈ I e x > a. Aﬁrmamos que f(x) → L quando x → a+. De fato, dado qualquer 𝜀 > 0, existe δ > 0 tal que L − 𝜖 < f(a + δ) ≤ L. Agora, como f é não crescente, para todo x ∈ I, 0 < x − a < δ, temos L − 𝜖 < f(a + δ) ≤ f(x) ≤ L e, portanto, |f(x) − L| < 𝜀. Os outros casos são análogos e deixados para o leitor (veja, também, Exercício 3.3.1).

Exercícios

E 3.3.1. Demonstre o Teorema 3.3.1 para o caso de uma função crescente e a ponto de acumulação à esquerda.
