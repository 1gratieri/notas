5.1 Derivada

Deﬁnição 5.1.1. (Derivada) Dizemos que uma função f : D → ℝ, y = f(x), é derivável (ou diferenciável) no ponto x = x0 ∈ D, se existe o limite da razão fundamental

f(x) − f(x0) x − x0 (5.1)

quando x → x0. Neste caso, o valor do limite é chamado de derivada da função f no ponto x0 e denotado por f′(x 0), Df(x0) ou df dx(x0).

Exemplo 5.1.1. Para f(x) = x temos f′(2) = 1∕(22). De fato,

lim x→2x −2 x − 2 = lim x→2 x − 2 (x − 2)(x + 2) = 1 22. (5.2)

Observação 5.1.1. Observemos que

f′(x 0) = lim x→x0f(x) − f(x0) x − x0 = lim h→0f(x0 + h) − f(x0) h (5.3)

usando a mudança de variável x = x0 + h.

Deﬁnição 5.1.2. (Função derivada) Dizemos que f : D → ℝ é uma função derivável em todo o seu domínio (ou em toda parte) quando f é derivável em todos os pontos de seu domínio. Neste caso, deﬁnição a função derivada de f por f′ : D → ℝ, y = f′(x), com

f′(x) = lim h→0f(x0 + h) − f(x0) h . (5.4)

Observação 5.1.2. A derivada lateral à direita ou à esquerda são deﬁnidas a partir da noção de limite lateral por

D+f(x0) = lim h→0+f(x0 + h) − f(x0) h (5.5)

e

D−f(x0) = lim h→0−f(x0 + h) − f(x0) h , (5.6)

respectivamente. Além disso, é imediato que Df(x0) existe se, e somente se, existem e são iguais as derivadas laterais D+f(x0) e D−f(x0).

Teorema 5.1.1. Toda função derivável num ponto x0 é contínua nesse ponto.

Demonstração.

Seja f : D → ℝ, y = f(x), uma função derivável no ponto x0 ∈ D. Vamos mostrar que f(x) − f(x0) → 0 quando x → x0. De fato, para x≠x0 temos

f(x) − f(x0) = f(x) − f(x0) x − x0 (x − x0). (5.7)

Agora,

lim x→x0f(x) − f(x0) x − x0 = f′(x 0) ∈ ℝe lim x→x0x − x0 = 0, (5.8)

logo f(x) − f(x0) → 0 quando x → x0.

■

Deﬁnição 5.1.3. (Diferencial) A diferencial de uma função derivável f no ponto x0 é o produto dy := f′(x 0)Δx, onde Δx = x − x0.

Observação 5.1.3. De sorte que o diferencial da função identidade y = x é dx = Δx e, portanto, o diferencial de uma dada função y = f(x) é dy = f′(x 0)dx e, também f′(x 0) = dy∕dx.
