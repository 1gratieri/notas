4.2 Propriedades de funções contínuas

Teorema 4.2.1. Teorema do valor intermediário Seja f : D → ℝ uma função contínua no intervalo fechado I = [a,b] ⊂ D, com f(a)≠f(b). Então, dado qualquer d compreendido entre f(a) e f(b) (inclusive), existe c ∈ I tal que f(c) = d.

Demonstração.

1.
Primeiramente, notamos que o resultado é imediato para os casos de d = f(a) e de d = f(b).
2.
Suponhamos f(a) < 0 < f(b) e, mostraremos que se d = 0, então existe c ∈ (a,b) tal que f(c) = d. Para tanto, usaremos o método da bisseção. Seja I(1) := [a(1),b(1)] = [a,b], l(1) o comprimento do intervalo I(1) e p(1) o ponto médio deste. Se f(p(1)) = 0 temos demonstrado o que queríamos. Agora, se f(p(1)) > 0, escolhemos I(2) = [a,p(1)]. Entretanto, se f(p(2)) < 0, escolhemos I(2) = [p(1),b]. Em qualquer um dos casos I(2) := [a(2),b(2)] ⊂ I(1), l(2) = l(1)∕2 e f(a(2)) < 0 < f(b(2)). Com isso, repetimos o procedimento de bisseção para o intervalo I(2) com p(2) o ponto médio deste. Se f(p(2)) = 0 temos o resultado desejado, caso contrário escolhemos o intervalo fechado I(3) := [a(3),b(3)] ⊂ I(2), l(3) = l(2)∕2 e f(a(3)) < 0 < f(b(3)). No pior dos casos, repetimos inﬁnitamente este procedimento e, com isso, temos construído uma sequência de intervalos fechados I(1) ⊂ I(2) ⊂ I(3) ⊂⋯ ⊂ I(n) ⊂⋯ cujos comprimentos tendem a zero. Logo, pelo Teorema dos intervalos encaixados I(1) ∩ I(2) ∩ I(3) ∩⋯ ∩ I(n) ∩⋯ = {c}⊂ I, o qual é o limite da sequência a(n) e da b(n). Daí, da continuidade da f e do fato de f(a(n)) < 0 < f(b(n)) temos
0 ≥ lim f(a(n)) = f(c) = lim f(b(n)) ≥ 0, (4.5)

donde segue que f(c) = 0, como queríamos demonstrar.

3.
Suponhamos que f(a) < f(b) e d ∈ (f(a),f(b)). Neste caso, tomamos g(x) = f(x) − d e, portanto, temos g(a) < 0 < g(b). Pelo demonstrado no item 2., existe c ∈ [a,b] tal que g(c) = 0 e, por consequência, f(c) = d.
4.
No caso de f(a) > f(b), tomamos g(x) = −f(x), de forma que g(a) < g(b). Então, pelo item 3., temos o resultado desejado.

Lema 4.2.1. Toda função contínua f : I = [a,b] → ℝ é limitada.

Demonstração.

Demonstraremos por absurdo. Seja f : I = [a,b] → ℝ uma função contínua não limitada. Denotemos I(1) := I. Como f é não limitada em I(1), temos que f é não limitada em pelo menos uma das metades do intervalo I(1). Seja, então I(2) uma das metades de I(1) na qual f é não limitada. Sucessivamente, construímos uma sequência de intervalos fechados I(n) nos quais f é ilimitada e cujos comprimentos tendem a zero. Então, pelo Teorema dos intervalos encaixados, existe um c ∈ I(1) ∩ I(2) ∩ I(3) ∩⋯ ∩ I(n) ∩⋯ ⊂ I. Agora, pela continuidade de f, temos que f(x) → f(c) quando x → c e, portanto, existe δ > 0 tal que x ∈ V δ(c) implica f(c) − 1 < f(x) < f(c) + 1, i.e. f é limitada no intervalo (c − δ,c + δ). Mas, como I(n) ⊂ (c − δ,c + δ) para n suﬁcientemente grande, temos f limitada em I(n), o que é um absurdo.

Teorema 4.2.2. Toda função contínua f : I = [a,b] → ℝ tem valor máximo e mínimo.

Demonstração.

Vamos, primeiramente, mostrar que f tem valor máximo em I. Por absurdo, seja f : I = [a,b] → ℝ função contínua, M seu supremo (pelo Lema 4.2.1, f(I) é um conjunto limitado) e f(x) < M para todo x ∈ I. Então, 1∕(M − f(x)) é uma função positiva e contínua em I. Seja, então, M′ > 0 seu supremo (novamente garantido pelo Lema 4.2.1) e, então, para todo x ∈ I temos

1 M − f(x) ≤ M′ ⇒ f(x) ≤ M − 1 M′, (4.6)

o que é um absurdo, pois isto contradiz o fato de M ser o supremo de f(I). Logo, existe algum x ∈ I tal que f(x) = M. Analogamente, seja m o ínﬁmo de f(I). Então, − m é o supremo da função g(x) = −f(x) no intervalo I. Pelo que acabamos de demonstrar, existe x ∈ I tal que g(x) = −m e, por consequência, f(x) = m.

■

Teorema 4.2.3. Se f : I = [a,b] → ℝ é uma função contínua, então f(I) é um intervalo limitado e fechado.

Demonstração.

Do Teorema 4.2.2 sejam m e M os valores mínimo e máximo de f, respectivamente. Logo, f(I) ⊂ [m,M]. Agora, sejam c,d ∈ I tal que f(c) = m e f(d) = M. Pelo Teorema do valor intermediário, dado qualquer d ∈ [m,M] existe x ∈ I tal que f(x) = d, i.e. d ∈ f(I). Portanto, [m,M] ⊂ f(I).
