6.1 Integral de Riemann

Deﬁnição 6.1.1. (Partição de um intervalo) Uma partição P de um intervalo [a,b] é um conjunto ordenado da forma

P([a,b]) = {a = x0 < x1 < x2 < … < xn = b}. (6.1)

O valor |P| = max 1≤i≤nΔxi, Δxi = xi − xi−1, é chamado de norma da partição.

Deﬁnição 6.1.2. (Reﬁnamento de uma partição) Um reﬁnamento de uma partição Pn([a,b]) := {a = x0,x1,x2,…,xn = b} é uma partição Pm([a,b]) com m > n tal que Pn([a,b]) ⊂ Pm([a,b]).

Deﬁnição 6.1.3. (Integral de Riemann) A integral de Riemann de uma função f : D → ℝ, y = f(x), num intervalo [a,b] ⊂ D, quando existe, é o número I tal que

I = lim n→∞∑ i=1nf(ξ i)Δxi, (6.2)

onde arbitrariamente ξi ∈ [xi−1,xi] e Δxi = xi − xi−1 são tomados considerando todas as possíveis partições P([a,b]) = {a = x0,x1,x2,…,xn = b}, com |P|→ 0 quando n → 0. Quando um tal I existe, dizemos que f é integrável em [a,b].

Observação 6.1.1. As somas parciais

Sn = ∑ i=1nf(ξ i)Δxi (6.3)

que aparecem na deﬁnição da integral de Riemann são chamadas de somas de Riemann.
